# How to Clone
* Fork this repo
* On your copy of the repo, select code, then copy the https link
* on your machine make a folder to store this project
* in your terminal, cd into the new folder that you just made
* type the command "git clone" + the URL, then press enter. Everything should be added to your folder now.
* Open the folder in VSCode
* Add something to the index.html file and commit that change
